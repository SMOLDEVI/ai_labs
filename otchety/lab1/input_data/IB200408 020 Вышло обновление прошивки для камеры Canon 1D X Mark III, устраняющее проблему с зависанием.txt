Вышло обновление прошивки для камеры Canon 1D X Mark III, устраняющее проблему с зависанием
   Вышло обновление прошивки для камеры Canon 1D X Mark III, устраняющее проблему с зависанием
Кроме того, оно добавляет возможность съемки видео с кадровой частотой 23,98 к/с
Как и было обещано, компания Canon выпустила обновление встроенного программного обеспечения для камеры 1D X Mark III. Прошивка версии 1.1.0 устраняет проблему с зависанием указанной камеры при определенных условиях. Помимо этого, в обновлении исправлены другие ошибки и сделаны некоторые улучшения.
Напомним, камера зависала при нажатии кнопки AF-ON или кнопки спуска затвора, если на дисплей видоискателя при этом выводилась индикация электронного уровня.
 
Также обновление устраняет проблему, из-за которой настройки «Пользовательского режима съемки» могут изменяться при переключении режима съемки, и добавляет возможность съемки видео с частотой 23,98 к/с. Кроме того, по словам Canon, повышена скорость связи при подключении через проводную локальную сеть, добавлена функция Browser Remote и возможность передачи только защищенных изображений.
Обновление прошивки доступно на сайте производителя.
