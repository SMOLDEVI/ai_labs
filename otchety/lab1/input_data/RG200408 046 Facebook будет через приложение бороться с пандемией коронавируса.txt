Facebook будет через приложение бороться с пандемией коронавируса
   Отследить, чтобы предотвратить, - вот так кратко можно описать основную идею новой инициативы Facebook "Data for Good" ("Данные во благо". - Прим. "РГ"). Компания намерена с помощью геолокации через приложение на смартфоне отслеживать перемещения пользователей в США для борьбы с пандемией коронавируса.
Эти данные передадут научным центрам и институтам здравоохранения, которые на их основе построят прогнозы. И все это, как уверяют в Facebook, на условиях полной анонимности и сохранности личных данных.

В Германии запустили "умное приложение" для диагностики коронавируса
Всего "продуктов" от таких наблюдений будет три. Карта совместного размещения позволит строить прогнозы по возникновению новых случаев COVID-19. Графики тенденций движения покажут, насколько соблюдается режим самоизоляции. А карта индекса социальной связанности на основе анализа друзей поможет определить, с какой вероятностью приятели из разных концов страны могут привезти друг другу COVID-19.
