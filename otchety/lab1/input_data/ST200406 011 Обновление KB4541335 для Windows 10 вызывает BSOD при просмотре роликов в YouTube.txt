Обновление KB4541335 для Windows 10 вызывает BSOD при просмотре роликов в YouTube 
Софт, 
Ноутбуки, 
IT-компании 


Согласно информации портала Windows Latest, после установки необязательного обновления KB4541335 для Windows 10 у некоторых пользователей возникло достаточное большое количество проблем с работоспособностью их ОС. Хотя это обновление должно было исправить некоторые ошибки и недочеты, которые сделали разработчики из Microsoft в прошлых обновлениях для Windows 10.

Проблемы, с которыми столкнулись пользователи, после установки обновления KB4541335 для Windows 10 (версии 1903 и 1909):


при просмотре в Chrome 80 видеороликов на YouTube появляется синий экран с ошибкой “whea_uncorrectable_error”; 
нестабильно работают некоторые игры — запускаются, но потом вылетают с ошибками; 
возникает очень высокая загрузка процессора ноутбука в режиме зарядки АКБ; 
интерфейс ОС неправильно стал отрабатывать работу с окнами — изменение размера окон, попытки нажать Restore Down или Restore Up не работают как в штатных приложениях (Windows Explorer, браузер Microsoft Edge и панели управления), так и в Microsoft Store (настройки, Microsoft Money, Flipboard). 

В настоящее время в компании Microsoft не будут отзывать или исправлять это обновление из-за этих ошибок, так как оно не является обязательным для установки, а проблемы зафиксированы только у ограниченной группы пользователей. Специалисты компании рекомендуют удалить это обновление и перезагрузить ОС пользователям, у которых стали возникать подобные ошибки после установки KB4541335.

Ранее 30 марта 2020 года Microsoft выпустила обновления: KB4554364 для ОС Windows 10 версий 1909 и 1903, KB4554354 для ОС Windows 10 версии 1809, KB4554349 для Windows 10 версии 1803, KB4554342 для Windows 10 версии 1709. Данные патчи исправляют проблему с сетевым соединением, которая начала возникать у части пользователей после установки февральского кумулятивного обновления KB453599. 

24 марта 2020 года Microsoft сообщила, что в компании решили начиная с мая 2020 года приостановить выпуск всех необязательных обновлений (так называемые обновления C и D), которые не связаны с безопасностью программных продуктов, для всех поддерживаемых в настоящее время клиентских и серверных версий ОС Windows. Таким образом, Microsoft устанавливает максимальный приоритет на безопасности, чтобы в текущей ситуации обеспечить стабильность, надежность и продуктивность всех версий ОС Windows, установленных у пользователей.

20 марта 2020 года Microsoft объявила в своем блоге, что компания приостанавливает разработку стабильной версии нового браузера Edge под номером 81, в связи со сложными текущими проблемами у сотрудников и возникшими «глобальными обстоятельствами».

19 марта 2020 года Microsoft объявила, что продлевает на шесть месяцев дату окончания поддержки ОС Windows 10 версии 1709 для корпоративного и образовательного сегмента (Windows 10 Enterprise, Education и IoT Enterprise). Теперь обновления безопасности для Windows 10 версии 1709 будут выходить до 13 октября 2020 года. Ранее срок окончания поддержки этой версии ОС для корпоративных пользователей и образовательных учреждений был установлен на 14 апреля 2020 года.

12 марта Microsoft экстренно выпустила патч KB4551762 для Windows 10 version 1903 и version 1909, а также Windows Server version 1903 и version 1909. Уязвимость CVE-2020-0796 в протоколе сетевой коммуникации Microsoft Server Message Block 3.1.1 (SMBv3) позволяет взломать SMB-сервер и SMB-клиент под управлением Windows 10 версии 1903, Windows Server версии 1903 (установка Server Core), Windows 10 версии 1909 и Windows Server версии 1909 (установка Server Core).
