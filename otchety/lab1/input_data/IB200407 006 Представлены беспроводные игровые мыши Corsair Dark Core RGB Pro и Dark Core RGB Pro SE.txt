   Представлены беспроводные игровые мыши Corsair Dark Core RGB Pro и Dark Core RGB Pro SE
Модель Dark Core RGB Pro SE поддерживает беспроводную зарядку
Компания Corsair сообщила о выпуске игровых мышей Dark Core RGB Pro и Dark Core RGB Pro SE. Для отслеживания перемещений в них используется оптический датчик PixArt PAW3392 разрешением 18 000 точек на дюйм, а для подключения к компьютеру — беспроводный интерфейс. Модель Dark Core RGB Pro SE, кроме того, поддерживает беспроводную зарядку Qi.
 
В описании новинок производитель отмечает возможность выбора между фирменным интерфейсом Slipstream Wireless, обеспечивающим задержку менее 1 мс, Bluetooth и USB. Частота опроса достигает 2000 Гц.
 
К другим достоинствам Dark Core RGB Pro отнесены девять зон подсветки RGB, восемь программируемых кнопок и поддержка макрокоманд. В мышах используются переключатели Omron, рассчитанные на 50 млн срабатываний. Время работы без подзарядки составляет 50 ч. Встроенной памяти достаточно для сохранения трех профилей подсветки и макросов.
Продажи уже начались. Рекомендованная цена модели Corsair Dark Core RGB Pro равна 80 долларам.
